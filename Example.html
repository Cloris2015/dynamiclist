Cloris' dynamic list manipulation example

This page uses HTML 5, CSS, jQuery 1.10 and JSON. When the page loads, it uses jQuery ajax method $.getJSON() to retrieve JSON data from a text file on the server, and stores it in a variable.

Each time the refresh button is pushed, the onclick handler gets an entry from the data and appends it to the list using jQuery. The reset button clears up the list.
